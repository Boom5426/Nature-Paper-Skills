# Source Data Checks

Use this file when verifying whether the manuscript's figures, tables, and supplement expose enough underlying data.

## Minimum Checks

- Every major figure or table has a clear underlying data route:
  - repository
  - source-data file
  - supplement
  - reused public dataset record
- The `Data Availability` statement is consistent with what the figures and tables actually require.
- If the paper claims that all supporting data are in the supplement, verify that the source data really cover the plotted values or summary statistics.
- Reused datasets are separated from newly generated datasets in the wording.
- Restricted datasets still expose enough metadata or derived support for an editor to judge the paper's transparency.

## Common Failure Modes

- The statement says `all data are available`, but figure source data are missing.
- The statement names a repository but not the dataset family that lives there.
- The statement gives one access route for both generated and reused data even though they differ.
- The statement relies on `reasonable request` without a real institutional route.
- The source-data files cover summary plots but not the underlying comparisons that matter to the claim.
